# react-native-quick-base64

A native implementation of base64 in C++ for React Native

## Installation

```sh
npm install react-native-quick-base64
```

## Usage

```js
import { multiply } from 'react-native-quick-base64';

// ...

const result = await multiply(3, 7);
```
